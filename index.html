Quiz WebApp

A timed web application that tests the users' knowledge on various topics through multiple-choice quizzes. It has a user registration and login system. It has 5 different levels of difficulties and their respective leaderboard.

The backend was Build using Spring Boot 3 Framework. The project has its own database built using PostgreSQL, utilizing the Spring Data JPA. User authentication was implemented using Spring Security 6.

The frontend was built using HTML, CSS, and Bootstrap 5 using Thymeleaf as a template engine. Software testing was done using JUnit 5 ,Mockito 3, and Spring Security Test.

Board Game

The GameBoard project is a Java application that implements a strategic board game. It features a game board with a size of 5 rows and 5 columns, where two players compete against each other using a set of 7 stones each.

Java is core programming language used to build the application. The JavaFX library is utilized for creating the graphical user interface of the game, allowing for a visually engaging user experience.

Maven is utilized as the build and dependency management tool for the project, streamlining the development process. The JUnit Jupiter framework is employed for writing unit tests to ensure the correctness of the application's functionality.

FillChecker App

A Forge app written in Javascript that checks Team members work log hours in a Jira global page. The app queries Forge Rest API for work logs and displays the logged and missing hours for team members.

It's built and deployed using the Forge platform within the Atlassian ecosystem. It utilizes Forge UI. Docker was used to build, test and deploy the app to the platform.

Website Automated Testing

An automated testing project using Selenium WebDriver , Java, and Cucumber. The project is a test suite for the SauceDemo website.

It uses Cucumber for behavior-driven development (BDD) testing. Utilizes Selenium WebDriver for automating web browser interactions. It also uses JUnit for unit testing.
